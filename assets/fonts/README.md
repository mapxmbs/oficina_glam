# Fontes do Projeto Oficina Glam

Para que a aplicação funcione corretamente, você precisa adicionar as seguintes fontes nesta pasta:

## 1. LOVELO BLACK
Usada exclusivamente em CAIXA ALTA para chamadas e informações de destaque.

**Como obter:**
- Baixe de: https://www.dafont.com/lovelo.font
- Extraia e renomeie para: `LOVELOBL.otf`

## 2. MONTSERRAT ALTERNATES MEDIUM
Aplicada em textos corridos e informações complementares.

**Como obter:**
- Baixe de: https://fonts.google.com/specimen/Montserrat+Alternates
- Selecione o peso "Medium (500)"
- Renomeie para: `MontserratAlternates-Medium.ttf`

## 3. INTER REGULAR
Aplicada em textos corridos e informações complementares, garantindo legibilidade.

**Como obter:**
- Baixe de: https://fonts.google.com/specimen/Inter
- Selecione o peso "Regular (400)"
- Renomeie para: `Inter-Regular.ttf`

## Estrutura esperada:
```
assets/fonts/
  ├── LOVELOBL.otf
  ├── MontserratAlternates-Medium.ttf
  ├── Inter-Regular.ttf
  └── README.md
```

## Hierarquia Tipográfica:
- **LOVELO BLACK**: Títulos, chamadas e destaques (sempre em CAIXA ALTA)
- **Montserrat Alternates Medium**: Textos secundários e informações importantes
- **Inter Regular**: Texto base e informações complementares

Após adicionar as fontes, reinicie o servidor Expo com `npx expo start -c`
